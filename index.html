⚠️ پێشەکی (تایبەت ب لینکێ تیک تۆک و اینستگرامێ):

  1. دەمێ لینکێ سایتی ژ TikTok یان Instagram ڤەدکەی، سەرەتا تبلأ خۆ ل دوگمەیا سەری بدە (Open in Browser یان نیشانا ڤەکرنێ د بروسەرێ دا).
  2. ئەڤە گەلەک گرنگە، چنکو سۆشیال مێدیا ڕێگریێ ل هندەک تایبەتمەندیێن بروسەرا فەرمی دکەت و تو نەشی سایتی ل هۆم سکرینێ داخل بکەی تا نەچیە بروسەرا فەرمی.

بۆ ئایفۆنی (iPhone - Safari):

  1. پشتی سایتی ب Safari ڤەکری، کلیک ل سەر دوگمەیا Share (نیشانا ڤەگوهاستنێ ل خوارێ) بکە.
  2. بگەڕە ل دووڤ Add to Home Screen و کلیک بکە.
  3. کلیک ل سەر Add ل سەرێ ڕاست بکە تا ل سەر شاشەیێ وەک ئاپەک پەیدا بیت.

بۆ ئەندرویدی (Samsung / Chrome):

  1. پشتی سایتی ب Chrome یان Samsung Internet ڤەکری، کلیک ل سەر سێ خاڵێن Menu ل سەرێ ڕاست بکە.
  2. Add to Home screen یان Install app هەلبژێرە.
  3. کلیک ل سەر Add / Install بکە دا ڕاستەوخۆ بێتە سەر شاشەیا موبایلا تە.
